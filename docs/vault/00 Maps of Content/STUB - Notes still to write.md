## How to use this

When you wiki-link something that doesn't exist yet, Obsidian shows the link in a different color. You can either:

1. Create the note immediately (click the link)
2. Add it here as a `[ ]` checkbox so you don't lose track

Periodically (weekly?) come here, pick the most useful one, and write it.

Don't try to clear this list. It's normal for it to grow and shrink. The goal is not zero stubs; the goal is "no important concept is wiki-linked but never written."
